# sales-analysis-project
This project analyzes sales data using Python and displays insights through an interactive dashboard built with Streamlit.

## Tools Used
- Python
- Pandas
- plotly
- Streamlit
- Excel

## Project Files
- sales.py : Python code for analysis and dashboard
- sales analysis.xlsx : Dataset used for analysis

## Goal
The goal of this project is to analyze sales performance and visualize insights through an interactive Streamlit dashboard.
